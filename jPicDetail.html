jPicDetail

List of Objects with hidden details

jQuery-Plugin, tested with 1.6x, all Markup HTML 5. Ignoring any prehistoric browser. Deal with it. So use recent versions of Firefox, Opera, Safari or Chrome. Btw. Fuck IE.

Hover over an object to slide in the details

Parameters

				
$('.jPicDetail').jPicDetail( {
	stickDurationAfter: 0,	// Stick duration after leaving the image
	overlayColor: "#000",	// Overlay Color
	overlayTransparency: 1,	// Transparancy of the background
	descrTextColor: "#FFF",	// Color of the description text
	descriptionPadding: 10	// Set textpadding, don't use css
 });
